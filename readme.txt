Ni CRM Lead is a WordPress plugin for managing customer inquiries across services and products.

== Description ==

Ni CRM Lead is a straightforward WordPress plugin designed to streamline lead creation. It efficiently maintains customer or vendor information along with follow-up history. This plugin enables you to easily store customer inquiries for various services and products, enhancing organization and efficiency.


Some Features.

* Fast And Easy Install
* Add New Customer
* Add New Inquiry 
* Add Update, Delete Follow Up
* Add new Follow up
* Add, Delete, Service, Product and status
